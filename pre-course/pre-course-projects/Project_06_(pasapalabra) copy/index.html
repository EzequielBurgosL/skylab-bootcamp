Welcome to Skylab's PASAPALABRA Project!

Let's review GAME INSTRUCTIONS:

There are 27 questions. Each question matches with one letter of the spanish alphabet. The general purpose of the game is to guess which word contains or begins with that letter. You can introduce the word pressing 'Enter answer' button. In case you are dubious about the right answer, type 'PASAPALABRA' (which stands for 'next word' in spanish) and leave the word-guess for the next circle round.

Finally, press 'Refresh' button to restart the game once you've completed it.
